# Skills 目录

AI 代码生成模板库，用于指导 Claude 生成符合项目规范的代码。

## 技能列表

### React 技术栈

| 技能 | 用途 |
|------|------|
| `form-generator-react` | 表单组件 (RHF + Zod + AntD) |
| `crud-template-react` | CRUD 模块模板 |
| `api-layer-react` | API 层封装 (Axios + React Query) |
| `unit-test-react` | 单元测试 (Jest + Testing Library) |
| `code-standard-react` | 代码规范标准 |
| `component-wrapper-react` | UI 组件封装 |

### Vue3 技术栈

| 技能 | 用途 |
|------|------|
| `form-generator-vue3` | 表单组件 (VeeValidate + Element Plus) |
| `crud-template-vue3` | CRUD 模块模板 |
| `api-layer-vue3` | API 层封装 (Axios + Vue Query) |
| `unit-test-vue3` | 单元测试 (Vitest) |
| `code-standrad-vue3` | 代码规范标准 |
| `component-wrapper-vue3` | UI 组件封装 |

## 使用方式

将技能文件内容复制到 Claude 对话中，或配置到 Claude Code/Deskop 的 skills 目录。

## 文件命名规范

```
{功能名}-{框架}.skill.md
```

示例: `form-generator-react.skill.md`
